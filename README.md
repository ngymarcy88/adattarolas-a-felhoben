# adattarolas-a-felhoben

A minimal Node.js application scaffold.

## Requirements

- Node.js 18 or newer
- npm

## Scripts

- `npm start` runs the application.
- `npm test` runs the Node.js test suite.

## Usage

```bash
npm install
npm start
```

Set `NAME` to customize the greeting:

```bash
NAME="Node.js" npm start
```
