# Product dokumentumok

Ez a mappa tartalmazza a 2. mérföldkőben rögzített termékvíziót, MVP-határokat, képességeket, tervezett sikermetrikákat és fő UX-folyamatokat. A dokumentumok tervezett működést írnak le; a funkciók még nem tekinthetők implementáltnak.

- [Termékvízió](vision.md): probléma, célfelhasználók, értékajánlat és termékalapelvek.
- [MVP scope contract](scope_contract.md): user story-k, acceptance criteria, korlátok és non-goals.
- [Capability map](capability_map.md): value és productization képességek MVP/későbbi bontásban.
- [Termékmetrikák](metrics.md): tervezett célértékek és verifikációs módszerek.
- [Fő UX-folyamatok](ux_flows.md): feltöltés, használat/megosztás és törlés/visszaállítás.
